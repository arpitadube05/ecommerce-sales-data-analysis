# E-Commerce Sales Data Analysis

## Project Overview
Analyzed e-commerce sales data using Python and SQL to identify monthly sales trends, top-performing products, and revenue patterns.

## Tools Used
Python, Pandas, NumPy, SQL, Matplotlib, Seaborn

## Analysis Performed
- Data cleaning and preprocessing
- Monthly sales trend analysis
- Top-performing products identification
- Revenue analysis with visualizations

## Key Insights
- Identified peak and low sales months
- Analyzed product-wise revenue contribution

## Conclusion
This project demonstrates practical data analysis skills using Python and SQL on real-world sales data.
